High-Performance Computing

  • Tricolor — Distributed HPC Cluster

    CPSC-375 High-Performance Computing — April 2026

    Built a 3-node Beowulf-style cluster from scratch, named after Venezuelan national symbols: Araguaney, Turpial, and Orquidea (3× Intel i7-10700 / 16 GB, Ubuntu Server 22.04), coordinated from a macOS head node. Configured passwordless SSH in all directions, an NFS shared filesystem, and OpenMPI across the private 1 GbE network. Benchmarked the cluster with HPL, achieving a peak of 242.01 GFLOPS — a 97.6% improvement over the single-node baseline — through strong-scaling and block-size parameter sweeps (N up to 60,000, NB up to 384, 48 MPI processes).

  • Parallel FIR Filter Architectures Using MPI

    CPSC-375 High-Performance Computing — Spring 2026

    Implemented and benchmarked three FIR filter architectures in distributed-memory MPI, modelled after the sequential, semi-parallel, and fully parallel designs from the Xilinx XtremeDSP guide. The semi-parallel version uses coefficient decomposition with MPI_Reduce; the parallel version uses signal (data) decomposition with halo exchange. Ran strong-scaling experiments on Trinity's Pine cluster (SLURM, up to 192 MPI processes across 8 nodes) at N=128 taps and L=107 samples, with correctness verified against a binary reference output from the serial baseline.

Embedded Systems & Electronics

  • Real-Time UART GUI PyQt5 for Photon Coincidence

    Undergraduate Research Project — Summer 2025

    Developed a multithreaded PyQt5-based desktop GUI for monitoring 15-channel photon coincidence events from a custom FPGA, using UART communication and pyqtgraph for high-speed plotting. Implemented live numerical displays, adjustable coincidence windows, event plotting with colour customisation, and automated CSV logging with buffered autosave every one second. Similarly, sends the coincidence window length to the FPGA via UART, and displays the coincidence count in real-time.

  • MOSPRO LIV:

    Capstone Project for Technician Degree in Electronics — Jan 2021 - Apr 2021

    MOSPRO LIV is a prototype of a car driven by a phone application. This was my capstone project to graduate as an industrial electronics mid-level technician. The car prototype consisted of a small car made by recyclable materials, an Arduino Uno as control unit, an HC-05 Bluetooth module, and two 5-volt DC motors. For direction control, we developed our own H-bridges with 2n3904 transistors and 1N4007 diodes.
