  • Real-Time UART GUI PyQt5 for Photon Coincidence

    Undergraduate Research Project — Summer 2025

    Developed a multithreaded PyQt5-based desktop GUI for monitoring 15-channel photon coincidence events from a custom FPGA, using UART communication and pyqtgraph for high-speed plotting. Implemented live numerical displays, adjustable coincidence windows, event plotting with colour customisation, and automated CSV logging with buffered autosave every one second.

  • DC Motor PID Speed Control System

    Embedded Systems Design project — May 2025

    Designed and implemented a PID-based PWM controller on Arduino UNO R4 for closed-loop DC motor speed regulation. Integrated encoder feedback and displayed motor speed (in RPM) in real time on a 16×2 LCD.

  • Heartbeat Monitoring System

    Embedded Systems Design project — Apr 2025

    Developed an 8051-based heartbeat monitor with an Assembly program that leverages 10-ms timer overflows to measure BPM from interrupt-driven pulse intervals as inputs. Programmed using Assembly language and hardware interfacing with a 7-segment display hardwired on a board.

  • Automatic Conveyor Belt for Sorting:

    Intern project — May 2021 - Jul 2021

    As an intern at Mahroy, I helped develop an automatic system to sort garment. For this 2-month project, I implemented the PID algorithm to control the conveyor belt by developing C/C++ code in the Arduino Framework, using the ESP32Servo, PID_v1, and PMW libraries.

  • MOSPRO LIV:

    Capstone Project for Technician Degree in Electronics — Jan 2021 - Apr 2021

    MOSPRO LIV is a prototype of a car driven by a phone application. This was my capstone project to graduate as an industrial electronics mid-level technician. The car prototype consisted of a small car made by recyclable materials, an Arduino Uno as control unit, an HC-05 Bluetooth module, and two 5-volt DC motors. For direction control, we developed our own H-bridges with 2n3904 transistors and 1N4007 diodes.
